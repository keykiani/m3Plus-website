DOWNLOADABLE RESOURCES (PDFs, etc.)
====================================
Place PDF downloads and other resource files here.

File naming convention:
  event-slug.pdf

Examples:
  designing-ai-creativity.pdf
  full-stack-designer.pdf
  ai-experience-design.pdf

After adding a PDF, reference it in the event's Markdown file:
  pdfUrl: "/resources/designing-ai-creativity.pdf"
