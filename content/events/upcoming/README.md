# Upcoming Events — No Longer Used

Upcoming events are now managed entirely through **Luma**.

## How to add a new event

1. Go to [lu.ma/m3plus](https://lu.ma/m3plus) and sign in
2. Create a new event in the M3+ calendar
3. That's it — the event will appear automatically on the website's Events page

The Luma calendar embed on the Events page updates in real time. No code changes or file edits are needed.

---

## After an event happens — archiving

Once an event is over, add it to the archive so the community can access photos and resources:

1. Copy `content/events/archive/_template.md`
2. Rename it to match the event slug (e.g. `my-event-name.md`)
3. Fill in the fields — especially `photosDriveUrl` and `resourcesDriveUrl`
4. Commit and push to GitHub → Netlify will deploy automatically

See the template file for full field documentation.
